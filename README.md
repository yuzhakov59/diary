# Проект "Дневник(заметки)"

## Описание:

-это инструмент для фиксации мыслей, событий, эмоций и размышлений о жизни.
Он помогает структурировать внутренний хаос, лучше понимать себя,
анализировать поступки, отслеживать прогресс и сохранять ценные воспоминания.

## Установка:

1. Клонируйте репозиторий:
```
git@github.com:yuzhakov59/diary.git
```

2. Установите зависимости:
```
список в файле requirements.txt
```

## Использование:

1. - User стандартная модель(создание с подтверждением по емайл, редактирование, аутентификация пользователя).
2. - для записей в дневнике создание, редактирование, удаление.
2. - для записей в дневнике возможность поиска по заголовку и тексту
